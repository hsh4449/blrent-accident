# 📧 Gmail → 네이버 메일 알림 설정 가이드

상담 신청 → 구글시트 저장 → Gmail 발송 → 네이버 메일 알림!

---

## 🎯 **전체 과정 (10분)**

```
1. Google Sheets 만들기 (2분)
2. Apps Script 코드 입력 (3분)
3. 권한 승인 (2분)
4. 테스트 (1분)
5. 웹 앱 배포 (2분)
```

---

## 📝 **1단계: Google Sheets 만들기**

### **1-1. 새 시트 생성**
```
1. https://sheets.google.com 접속
2. "새 스프레드시트" 클릭
3. 이름: "비엘렌트카 상담신청"
```

### **1-2. 헤더 입력**
```
첫 번째 줄에 다음과 같이 입력:

A1: 신청시간
B1: 이름
C1: 연락처
D1: 차량종류
E1: 사고일자
F1: 문의내용
```

---

## 💻 **2단계: Apps Script 코드 입력**

### **2-1. Apps Script 열기**
```
1. 상단 메뉴 "도구" → "Apps Script" 클릭
2. 새 창이 열림
3. 기존 코드 전체 삭제 (Ctrl+A, Delete)
```

### **2-2. 코드 복사 붙여넣기**
```
1. gmail-script.js 파일 열기
2. 코드 전체 복사 (Ctrl+A, Ctrl+C)
3. Apps Script에 붙여넣기 (Ctrl+V)
4. 저장 (Ctrl+S)
```

**중요:** 이메일 주소는 이미 `blrentcar@naver.com`으로 설정되어 있습니다!

---

## 🔑 **3단계: 권한 승인 (중요!)**

### **3-1. 테스트 실행**
```
1. Apps Script 상단에서 함수 선택: "test"
2. "실행" 버튼 (▶) 클릭
```

### **3-2. 권한 승인**
```
1. "권한 검토" 팝업 나타남
2. 구글 계정 선택
3. "고급" 클릭
4. "프로젝트 이름(안전하지 않음)으로 이동" 클릭
5. "허용" 클릭
```

### **3-3. 테스트 확인**
```
1. 구글 시트 확인 → 테스트 데이터 추가되었는지
2. 네이버 메일 확인 → 이메일 도착했는지
```

## 🌐 **4단계: 웹 앱 배포**

### **4-1. 배포 설정**
```
1. Apps Script 우측 상단 "배포" 클릭
2. "새 배포" 선택
3. 설정:
   - 유형: 웹 앱 (⚙️ 클릭)
   - 실행 계정: 나
   - 액세스 권한: 모든 사용자
4. "배포" 클릭
```

### **4-2. URL 복사**
```
웹 앱 URL이 나타남:
https://script.google.com/macros/s/AKfycby.../exec

이 URL을 메모장에 복사!
```

## 🔗 **5단계: 홈페이지 연결**

### **index.html 수정**

```javascript
// 이 부분 찾기 (Ctrl+F로 "scriptURL" 검색):
const scriptURL = "https://script.google.com/macros/s/...기존URL...";

// 👇 4단계에서 복사한 URL로 교체:
const scriptURL = "복사한_웹앱_URL";
```

**저장!**

## ✅ **6단계: 최종 테스트**

### **실제 홈페이지에서:**
```
1. 홈페이지 열기
2. 상담 신청 폼 작성:
   - 이름: 테스트
   - 연락처: 010-0000-0000
   - 차량: 테스트
   - 내용: 테스트입니다
3. "상담 신청하기" 클릭
4. "상담 신청이 완료되었습니다!" 알림 확인
5. 구글 시트 확인 → 데이터 저장됨
6. 네이버 메일 확인 → 메일 도착!
```

## 🆘 **문제 해결**

### **Q: 메일이 안 와요!**
```
A: 
1. 네이버 스팸함 확인
2. Gmail 발신함 확인
3. Apps Script 로그 확인 (Ctrl+Enter)
4. 이메일 주소 확인: blrentcar@naver.com
```

### **Q: 구글시트에만 저장되고 메일은 안 와요!**
```
A: 
1. Gmail 권한 승인 다시 확인
2. test() 함수 다시 실행
3. 로그에서 에러 메시지 확인
```

### **Q: 권한 오류가 계속 나요!**
```
A: 
1. 구글 계정 보안 설정 확인
2. 2단계 인증 확인
3. 앱 비밀번호 필요 여부 확인
```

### **Q: 메일이 스팸으로 가요!**
```
A: 
1. 네이버 메일에서 "스팸 아님" 클릭
2. 발신자를 주소록에 추가
3. 이후부터는 받은편지함으로 옴
```
