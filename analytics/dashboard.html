전환율 = 해당 버튼 전화클릭 ÷ 순방문자. 꼬리표(data-cta) 적용 후 발생한 클릭만 구분됩니다. 그 이전 클릭은 미분류로 표시됩니다.

🚨 부정클릭·업자 의심 탐지

예산소진 의심 = 광고(검색키워드) 3회 이상 유입 + 전환 0 + 평균체류 5초 미만 · 쿠키세탁 의심 = 같은 IP에서 방문자ID 3개 이상(쿠키 지우고 재방문) · 업자 정탐 의심 = 재방문 + 데스크톱 + 스크롤 75%+ + 평균체류 60초+ + 전환 0. IP 기반 항목은 수집장치(/track) 적용 후 방문부터 잡힙니다. 의심일 뿐 단정 아님 — 네이버·구글 신고 시 증거로 활용하세요.
