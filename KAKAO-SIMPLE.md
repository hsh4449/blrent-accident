# 💬 카카오톡 알림 설정 (5분 완성!)

상담 신청 → **카카오톡으로 즉시 알림!**

---

## 🎯 **전체 과정 (5분)**

```
1. 카카오톡 오픈채팅방 만들기
2. Incoming Webhook 봇 추가
3. Google Apps Script 설정
4. 완료!
```

---

## 📱 **Step 1: 카카오톡 오픈채팅방 만들기**

### **1-1. 오픈채팅방 생성**
```
1. 카카오톡 앱 열기
2. 하단 # 아이콘 클릭 (또는 검색에서 "오픈채팅")
3. 우측 상단 "+" 버튼
4. "오픈채팅방 만들기"
5. 설정:
   - 채팅방 이름: "아레스렌트카 상담알림"
   - 공개: OFF (비공개)
   - 멤버: 나만
6. "확인" 클릭
```

---

## 🤖 **Step 2: Incoming Webhook 봇 추가**

### **2-1. 봇 추가하기**
```
1. 오픈채팅방 입장
2. 우측 상단 "≡" (메뉴) 클릭
3. "관리" 클릭
4. "채팅봇" 클릭
5. "Incoming Webhook" 검색
6. 추가 클릭
```

### **2-2. Webhook URL 복사**
```
1. "Incoming Webhook" 클릭
2. "Webhook URL" 복사 (중요!)
   예: https://hooks.kakaoi.ai/...
```

---

## 📝 **Step 3: Google Sheets 준비**

### **3-1. 새 시트 만들기**
```
1. https://sheets.google.com
2. 새 스프레드시트
3. 이름: "아레스렌트카 상담신청"
```

### **3-2. 헤더 입력**
```
첫 번째 줄에:
A1: 신청시간
B1: 이름
C1: 연락처
D1: 차량종류
E1: 사고일자
F1: 문의내용
```

---

## 💻 **Step 4: Apps Script 설정**

### **4-1. 코드 입력**
```
1. 도구 → Apps Script
2. 기존 코드 전체 삭제
3. kakao-webhook.js 파일 내용 복사
4. 붙여넣기
```

### **4-2. Webhook URL 입력**
```javascript
// 이 부분 찾기:
const KAKAO_WEBHOOK_URL = "YOUR_WEBHOOK_URL_HERE";

// 👇 복사한 Webhook URL로 교체:
const KAKAO_WEBHOOK_URL = "https://hooks.kakaoi.ai/...복사한URL...";
```

**저장!** (Ctrl+S)

---

## 🌐 **Step 5: 웹 앱 배포**

```
1. 우측 상단 "배포" 클릭
2. "새 배포" 선택
3. 설정:
   - 유형: 웹 앱
   - 실행 계정: 나
   - 액세스 권한: 모든 사용자
4. "배포" 클릭
5. "웹 앱 URL" 복사!
   예: https://script.google.com/macros/s/.../exec
```

---

## 🔗 **Step 6: 홈페이지 연결**

### **index.html 수정**

```javascript
// 이 부분 찾기:
const scriptURL = "https://script.google.com/macros/s/AKfycbz6J...";

// 👇 Step 5에서 복사한 URL로 교체:
const scriptURL = "복사한_웹앱_URL";
```

---

## 🧪 **Step 7: 테스트**

### **7-1. Apps Script 테스트**
```
1. Apps Script에서 "test" 함수 선택
2. 실행 클릭
3. 권한 승인
4. 카카오톡 오픈채팅방 확인!
```

### **7-2. 실제 테스트**
```
1. 홈페이지 접속
2. 상담 신청 폼 작성
3. 제출
4. 카카오톡 확인!
```

## 🆘 **문제 해결**

### **Q: Incoming Webhook 봇이 안 보여요!**
```
A: 카카오톡 앱 최신 버전으로 업데이트
   또는 검색창에 "Incoming Webhook" 직접 검색
```

### **Q: 메시지가 안 와요!**
```
A: 
1. Webhook URL이 정확한지 확인
2. Apps Script 로그 확인 (Ctrl+Enter)
3. test() 함수로 먼저 테스트
```

### **Q: 권한 오류가 나요!**
```
A: Apps Script 실행 시 권한 승인 필요
   "고급" → "안전하지 않은 앱 허용" 클릭
```
